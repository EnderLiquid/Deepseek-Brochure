# DeepSeek V4 中文宣传页

 Vue 3 单页宣传网页，用于介绍 DeepSeek-V4 预览版的百万上下文、Pro / Flash 双模型、Agent 能力、API 接入与开源入口。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
